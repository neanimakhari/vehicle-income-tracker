# Data Subject Request (DSR) Workflow

## Intake
- Capture request details and verify identity.
- Log request ID and timestamps.

## Assessment
- Determine request type (access, delete, rectify).
- Identify impacted systems and data stores.

## Fulfillment
- Collect data from systems.
- Apply deletion or rectification if requested.

## Response
- Provide response to requester within SLA.
- Document completion and evidence.

## Retention
- Store DSR records per data retention policy.
