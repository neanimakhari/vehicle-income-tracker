# Control Mapping (Template)

Use this document to map ISO controls to system artifacts and evidence.

## ISO 27001

- **A.5 Information Security Policies** → Policy documents, revision history.
- **A.6 Organization of Information Security** → Roles, responsibilities, org chart.
- **A.8 Asset Management** → Asset inventory, data classification policy.
- **A.9 Access Control** → IAM rules, RBAC configs, access reviews.
- **A.10 Cryptography** → TLS configs, at-rest encryption, key management.
- **A.12 Operations Security** → Runbooks, monitoring, backup/recovery policy.
- **A.14 System Acquisition/Development** → SDLC, secure coding, change management policy.
- **A.16 Incident Management** → IR policy, incident tickets.
- **A.17 Business Continuity** → Business continuity policy, DR plan, recovery tests.
- **A.18 Compliance** → Audit reports, regulatory mapping.

## ISO 27017/27018

- **Cloud shared responsibility** → Provider contracts, responsibilities matrix.
- **Tenant isolation** → Schema-per-tenant design, access guards.
- **PII handling** → Privacy policy, data retention, DSR processes.

## ISO 27701

- **Privacy management** → DPIA templates, lawful basis tracking.
- **Data subject rights** → DSR workflows and logs.
