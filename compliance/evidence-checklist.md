# Evidence Checklist (Template)

## Access Control

- [ ] RBAC matrix for platform and tenant roles
- [ ] MFA policy and enforcement evidence
- [ ] Access review logs (quarterly)
- [ ] Password policy published

## Data Security

- [ ] TLS configuration (ingress + internal)
- [ ] Database encryption at rest enabled
- [ ] Backup encryption + restore test logs
- [ ] Backup and recovery policy published

## Audit Logging

- [ ] Audit log schema + retention policy
- [ ] Sample audit event exports

## Change Management

- [ ] Deployment approval records
- [ ] Release notes with change impact
- [ ] Change management policy published

## Incident Response

- [ ] Incident response plan
- [ ] Incident simulation/runbook evidence

## Privacy

- [ ] Data retention policy
- [ ] Data subject request process + sample tickets

## Business Continuity

- [ ] Business continuity policy published
- [ ] DR runbooks maintained
- [ ] Recovery test reports

## Vendor Management

- [ ] Vendor inventory maintained
- [ ] Vendor security reviews completed annually
